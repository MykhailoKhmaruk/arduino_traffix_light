Arduino Traffic Light with ENC28J60 ethernet web control
===================================

## Libraries
* [EtherCard](https://github.com/njh/EtherCard) ENC28J60 ethernet arduino library. 

## Physical Installation

### ENC28J60 PIN Connections (Arduino UNO):

    VCC -   3.3V
    GND -    GND
    SCK - Pin 13
    SO  - Pin 12
    SI  - Pin 11
    CS  - Pin 10 

### Relay

	Red 	- Pin 5
	Yellow  - Pin 6
	Green 	- Pin 7
